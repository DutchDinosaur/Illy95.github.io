3AM Spiders

(Jan 2020)
To make this I made a piece of inverse kinematics code that controls every leg. Then I used a flocking algorithm to make all the spiders walk in clusters. Back when i made this i didn't know about KDtrees so it isn't as optimized as it could be.

    Tools used:
  • - Unity
  • - Maya
  • - Photoshop
  • - Photonengine

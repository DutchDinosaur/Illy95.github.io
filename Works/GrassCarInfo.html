GrassCar

(May 2019)
For this project I used unity's shadergraph to create simulated grass. The grass mesh was my fist time generating meshes using code. The game isn't using any kind of object pooling so it runs out of memory very quickly, but it is very graphicly optimized, since the grass is all contained in big meshes and all the fancy work is done in shaders.

    Controls:
  • movement - WASD
  • Upgrade menu - E

    Tools used:
  • - Unity
  • - Maya
  • - Photoshop
